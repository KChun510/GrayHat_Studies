1) docker run -it i386/ubuntu bash
Explantion: 
Run the docker container "i386/ubuntu", download if not on local. 
-it flag, is a combo -i "interactive mode" keeps the sdin open. -t, allocates a virtual terminal to the container
Bash, command opens a bash shell whithin the term

2) docker start -i <docker container id -or- name>
-i opens in interactive mode

3) docker stop <docker container id -or- name>
